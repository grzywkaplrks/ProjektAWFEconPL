## 📋 O Projekcie

Niniejszy raport analizuje **efektywność ekonomiczną polskiej opłaty cukrowej** (wprowadzonej w 2021 r.) jako systemowego narzędzia polityki zdrowia publicznego. Badanie obejmuje pełen cykl interwencji fiskalnej — od generowania wpływów budżetowych, przez finansowanie programów profilaktycznych, aż po redukcję kosztów leczenia powikłań otyłości i cukrzycy typu 2.

Projekt realizowany w ramach **programu praktycznego Promocja Zdrowia** przy Akademii Wychowania Fizycznego im. Polskich Olimpijczyków we Wrocławiu, z zastosowaniem metodologii **Social Return on Investment (SROI)** zgodnej ze standardami Social Value UK oraz narzędzia analitycznego **Business Model Canvas (BMC-9)**.

## 🏗️ Struktura Raportu — Business Model Canvas (BMC-9)

Raport analizuje podatek cukrowy przez pryzmat wszystkich 9 elementów modelu BMC:

| # | Element BMC | Główna teza |
|---|-------------|-------------|
| **1** | 🎯 Segmenty klienta | Dzieci i młodzież, dorośli 18–45 lat, grupy wrażliwe socjoekonomicznie, NFZ, POZ, samorządy |
| **2** | 💎 Propozycja wartości | Podwójna dywidenda fiskalna i zdrowotna — korzyści budżetowe i behawioralne wzajemnie się wzmacniają |
| **3** | 📦 Kanały dystrybucji | System podatkowy MF, Sanepid, NFZ, placówki POZ, programy szkolne i kampanie MZ |
| **4** | 🤝 Relacje z klientem | Edukacja żywieniowa, reformulacja produktów, regulacja reklamy, programy profilaktyczne |
| **5** | 💰 Strumień przychodów | Opłata stała 0,50 zł/l + opłata zmienna 0,05 zł/g cukru powyżej 5 g/100 ml |
| **6** | 🔧 Kluczowe zasoby | Aparat skarbowy, bazy danych epidemiologicznych, kadra POZ (35 500 lekarzy) |
| **7** | ⚙️ Kluczowe działania | Pobór daniny, alokacja do NFZ, monitoring zdrowotny, ewaluacja skuteczności |
| **8** | 🌐 Kluczowi partnerzy | MZ, MF, NFZ, WHO, NCBR, samorządy, producenci napojów |
| **9** | 📊 Koszty / SROI | Całkowity budżet interwencji vs. 9,2x wartość społeczna w horyzoncie 5 lat |

## 🔬 Metodologia

### Social Return on Investment (SROI)

Wskaźnik SROI obliczono zgodnie ze standardem **Social Value UK (2024)** przy następujących założeniach:

```
Wartość społeczna wytworzona:   29,5 mld zł (horyzont 5 lat)
Łączna wartość nakładów:         3,2 mld zł/rok × 5 = 16,0 mld zł
─────────────────────────────────────────────────────────────────
SROI = 29,5 / 3,2 = 9,2 : 1
```

Składowe wartości społecznej:
- 💊 Redukcja kosztów leczenia cukrzycy t.2, ChSN i NAFLD w NFZ (~1,4 mld zł/rok)
- 🏥 Oszczędności systemowe z programów profilaktycznych finansowanych z wpływów
- ⏳ Zysk lat życia w pełnym zdrowiu (QALY) — wycena metodą WTP
- 🏭 Wzrost produktywności populacji aktywnej zawodowo

### Business Model Canvas (BMC-9)

Adaptacja narzędzia Osterwaldera do analizy interwencji fiskalnej w systemie zdrowia publicznego — jako alternatywa dla klasycznej analizy kosztów-korzyści (CBA), uwzględniająca aspekty niematerialne i systemowe.

> ⚠️ **Nota o konflikcie interesów:** Sponsorzy z sektora gastronomicznego (McDonald's Polska, Heineken Polska, KFC Poland, Bodrum Kebab Radomsko) przekazali środki w formule *unrestricted educational grant* — bez wpływu na metodologię, wyniki i rekomendacje. Ironia wyboru sponsorów z branży objętej analizą jest **w pełni zamierzona** i stanowi element analizy konfliktów interesów w badaniach zdrowia publicznego *(Moodie et al., The Lancet, 2013)*.
